# v2.5.1 Release Notes

## Summary
This patch release focuses on internal improvements, build process updates, and minor fixes to enhance the project's stability and maintainability.

## Added
- DeepWiki badge.

## Changed
- Updated build scripts to use `package.json` for module type configuration (CommonJS/ESM) and streamlined the build process by replacing `rename-cjs.js` with `move-package-json.js`.
- Refactored `TimersStore` to utilize `fs/promises` for more efficient asynchronous file operations.

## Fixed
- Corrected file extensions in `package.json` from `.cjs` to `.js` for better compatibility.
- Resolved an issue with `replace` functionality.
- Updated the `flatted` dependency to its latest version.

## Removed
- Unused `dev` and `start` scripts from `package.json`.
- `nodemon` and `ts-node` development dependencies.
- The `rename-cjs.js` build script, replaced by `move-package-json.js`.

## Notes
- Internal code quality was improved through a linting pass.
