# v2.5.0 Release Notes

## Summary
This release introduces new functionalities for `JSONLTimersManager`, allowing users to modify timer titles and descriptions. It also includes internal improvements and dependency updates.

## Added
- `JSONLTimersManager` now includes `changeTitle(id: string, newTitle: string)` to update a timer's title.
- `JSONLTimersManager` now includes `changeDescription(id: string, newDescription: string)` to update a timer's description.

## Changed
- Internal dependency updates have been performed for improved stability and security.
- The return type for `createTimersStore` in `JSONLTimersManager` and `PlainTextTimersManager` has been made more specific for better type inference (internal change).
- The `runExclusive` method in `TimersManager` has changed its visibility from `private` to `protected`, allowing subclasses to access it.
